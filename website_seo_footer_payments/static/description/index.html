What's inside ?

This Odoo module removes Heading tags (Hn) around payment methods list in footer. This help Google to understand page meaning.

SEO Footer Payments

Better for search engines

Why do you need this module ?

Focus on page content

Odoo prints headings tags around payments method list in footer. Headings in footer breaks the page readibility for search engines, and confuse them on page meaning. Removing headings help search engines to focus on page content.

For complete SEO footer, you should also install SEO Footer (website_seo_footer)
